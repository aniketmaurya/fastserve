# FastServe

Machine Learning Serving focused on LLMs with simplicity as the top priority.

## Installation

```bash
git clone https://github.com/aniketmaurya/fastserve.git
cd fastserve
pip install .
```

## Run locally

```bash
python -m fastserve
```


## Usage/Examples

```python
from fastserve import FastServe

serve = FastServe()
serve.run_server()
```
